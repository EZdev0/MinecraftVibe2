# MinecraftVibe2
Mc vibecoding 2 Gemini No AIDE 

## UI Editor Optimierung
Der UI Editor wurde optimiert, um die Positionen aller Buttons (Angriff, Sprung, Sneak, Sprint etc.) zuverlässig zu speichern und wiederherzustellen.
Dies wurde durch eine dynamische Anpassung der Layout-Gravity erreicht, um Konflikte mit absoluten Koordinaten zu vermeiden. Jede Komponente nutzt nun eigene LayoutParams.
